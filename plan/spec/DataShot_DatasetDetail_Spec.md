## 1. 화면 개요

| 항목 | 내용 |
|---|---|
| 화면명 | 데이터셋 상세 (Dataset Detail) |
| 경로 | `/datashot/{id}` |
| 진입 경로 | DataShot 목록 → Completed 상태 행 클릭 |
| 접근 권한 | Admin, Marketer, Client, Agency (전 역할 조회 가능) |
| 진입 조건 | 데이터셋 상태가 `Completed`인 경우에만 접근 가능 |
| 주요 기능 | 메타데이터 조회, Raw 데이터 테이블(필터/정렬/페이지네이션), CSV Export, 복제/이동/삭제 |

## 2. 레이아웃 구조

### 2.1 전체 구조

```
┌─────────────────────────────────────────────────────┐
│ AppLayout (GlobalNavBar + Sidebar + Breadcrumb)      │
├─────────────────────────────────────────────────────┤
│ 헤더: 데이터셋명 | 메타 요약 | 액션 버튼             │
├─────────────────────────────────────────────────────┤
│ Data Limit Warning (조건부)                          │
├─────────────────────────────────────────────────────┤
│ Extracted Data 테이블 (필터 행 + 데이터 행)           │
├─────────────────────────────────────────────────────┤
│ 페이지네이션                                         │
└─────────────────────────────────────────────────────┘
```

### 2.2 의존 모듈

| 모듈 | 역할 |
|---|---|
| `AppLayout` | 전체 레이아웃 래퍼 (GNB + Sidebar + Breadcrumb) |
| `useSidebarState` | 사이드바 상태 관리 훅 |
| `maskEmail` | 이메일 마스킹 유틸 |
| `getDarkMode` / `setDarkMode` | 다크모드 상태 관리 |
| `IndustryModal` / `AdProductsModal` / `MetricsModal` | 읽기 전용 상세 모달 (DatasetDetailModals.tsx) |
| `generateSampleData` | 샘플 데이터 생성 (sampleData.ts) |

### 2.3 Breadcrumb

| 순서 | 라벨 | 동작 |
|---|---|---|
| 1 | SlotBoard | `/slotboard`로 이동 |
| 2 | {Slot명} | Slot 상세로 이동 |
| 3 | DataShot | `/datashot`로 이동 |
| 4 | {데이터셋명} | 현재 페이지 (비활성) |

## 3. 헤더 영역

### 3.1 데이터셋명

| 요소 | 스펙 |
|---|---|
| 텍스트 | 데이터셋명 (font-size: 20px, font-weight: 500, Paperlogy) |
| overflow | ellipsis (긴 이름 말줄임) |

### 3.2 메타데이터 요약 (인라인)

| 항목 | 표시 형식 | 클릭 동작 |
|---|---|---|
| 매체 | "매체 {매체명}" | 없음 |
| 조회기간 | "조회기간 {start} → {end}" | 없음 |
| 업종 | "업종 {N}개" + SearchCheck 아이콘 | IndustryModal 오픈 |
| 조회조건 | "조회조건 {N}개" + SearchCheck 아이콘 | AdProductsModal 오픈 |
| 지표 | "지표 {N}개" + SearchCheck 아이콘 | MetricsModal 오픈 |

- 각 항목은 `•` 구분자로 연결
- font-size: 12px, color: muted-foreground

### 3.3 액션 버튼 (우측)

| 버튼 | 아이콘 | 동작 |
|---|---|---|
| Share | Share2(16px) | Export 메뉴 드롭다운 오픈 |
| Info | Info(16px) | 정보 툴팁 표시 (hover) |
| More | MoreVertical(16px) | 컨텍스트 메뉴 오픈 |

#### 3.3.1 Export 메뉴 (Share 클릭)

| 메뉴 항목 | 아이콘 | 동작 |
|---|---|---|
| Copy Link | Link2(16px) | 현재 URL 클립보드 복사 → 성공 토스트 |
| Export to CSV | FileSpreadsheet(16px) | Full 파일 CSV 다운로드 |

- CSV 파일명: `DS_Dataset_Result_yyyymmdd.csv` (강제 삽입, 변경 불가. 예: DS_Dataset_Result_20260526.csv)
- 화면 필터/정렬 상태와 무관하게 Full 파일 전체 행 다운로드

#### 3.3.2 Info 툴팁 (hover)

| 표시 항목 | 내용 |
|---|---|
| 설명 | 데이터셋 설명 텍스트 |
| Dataset ID | #{id} |
| 생성일시 | YYYY-MM-DD HH:mm |
| 생성자 | {이름} ({마스킹된 이메일}) |

- width: 280px, position: absolute right

#### 3.3.3 컨텍스트 메뉴 (More 클릭)

| 메뉴 항목 | 아이콘 | 동작 | 권한 |
|---|---|---|---|
| 복제 | Copy(14px) | 조회 조건 상속 → 생성 화면 진입 | Admin, Marketer |
| 이동 | ArrowRightLeft(14px) | 이동 처리 | Admin, Marketer |
| 삭제 | Trash2(14px) | 삭제 확인 다이얼로그 오픈 | Admin, Marketer |

## 5. Extracted Data 테이블

### 5.1 테이블 구조

| 영역 | 설명 |
|---|---|
| 섹션 타이틀 | "Extracted Data" (font-size: 20px, font-weight: 500) |
| 필터 초기화 버튼 | 필터 적용 시에만 노출 (RefreshCcw 아이콘 + "필터 초기화") |
| 데이터 업데이트 정보 | "최근 데이터 업데이트: {날짜}" + Info 툴팁 |
| 테이블 | 헤더 행 + 필터 행 + 데이터 행 |
| 페이지네이션 | 테이블 하단 |

### 5.2 테이블 컬럼

#### 5.2.1 Meta 컬럼 (좌측, 텍스트 정렬: left)

| 컬럼 | 키 | 정렬 가능 |
|---|---|---|
| 기간 | period | O |
| 매체 | media | O |
| 업종(대) | industryLarge | O |
| 업종(중) | industryMedium | O |
| 업종(소) | industrySmall | O |
| 광고상품 컬럼 (매체별 동적) | 매체에 따라 다름 | O |
| 타겟팅 옵션 (선택 시) | targetingOption | O |

- Meta 매체 광고상품 컬럼: 캠페인 목표, 구매 유형, 플랫폼, 성과 목표
- 기타 매체: 광고상품 (1컬럼)

#### 5.2.2 지표 컬럼 (우측, 텍스트 정렬: right)

| 컬럼 | 키 | 정렬 가능 | 단위 |
|---|---|---|---|
| 광고비 | cost | O | 원 |
| 노출수 | impressions | O | - |
| 클릭수 | clicks | O | - |
| CTR | ctr | O | % |
| CPC | cpc | O | 원 |
| CPM | cpm | O | 원 |
| VTR | vtr | O | % |

- 지표 컬럼은 데이터셋 생성 시 선택한 지표에 따라 동적 구성
- 지표 컬럼 시작 시 좌측 border로 시각적 구분

### 5.3 정렬

| 동작 | 설명 |
|---|---|
| 헤더 클릭 | 해당 컬럼 asc 정렬 |
| 같은 헤더 재클릭 | asc → desc 토글 |
| 정렬 아이콘 | ArrowUp(asc) / ArrowDown(desc), 활성 컬럼에만 표시 |
| 숫자 컬럼 | 숫자 비교 정렬 |
| 텍스트 컬럼 | localeCompare 정렬 |

### 5.4 필터 행

테이블 헤더 바로 아래에 필터 행이 위치하며, 컬럼 타입에 따라 두 가지 필터 제공:

#### 5.4.1 목록형 필터 (Meta 컬럼)

| 요소 | 스펙 |
|---|---|
| 트리거 | 드롭다운 버튼 (기본: "전체") |
| 선택 표시 | 1개: 값 표시 / 2개+: "{N}개 선택" / 전체: "전체" |
| 드롭다운 | width: 240px, max-height: 320px |
| 검색 | 드롭다운 내 검색 input |
| 전체 선택 | 체크박스 (전체 선택/해제 토글) |
| 옵션 목록 | 체크박스 다중 선택 |
| 초기화 | 선택값 있을 때만 "초기화" 버튼 노출 |
| 필터 적용 시 | currentPage = 1로 리셋 |

| 대상 컬럼 |
|---|
| period, media, industryLarge, industryMedium, industrySmall, objective, buyingType, platform, performanceGoal, targetingOption |

#### 5.4.2 지표 필터 (Metric 컬럼)

| 요소 | 스펙 |
|---|---|
| UI | 부등호 select + 숫자 input (하나의 박스) |
| 부등호 옵션 | =, >, <, ≥, ≤ |
| 숫자 입력 | 콤마 포맷팅 (1,000 단위), 숫자만 허용 |
| 필터 적용 | operator + value 모두 있을 때만 적용 |
| 필터 적용 시 | currentPage = 1로 리셋 |

### 5.5 필터 초기화

| 요소 | 스펙 |
|---|---|
| 노출 조건 | 목록형 필터 또는 지표 필터 중 하나라도 값이 있을 때 |
| 버튼 스타일 | bg: foreground, color: background, RefreshCcw 아이콘 |
| 동작 | 모든 필터 초기화 + currentPage = 1 |

### 5.6 데이터 업데이트 정보

| 요소 | 스펙 |
|---|---|
| 텍스트 | "최근 데이터 업데이트: {날짜}" (font-size: 11px) |
| Info 툴팁 | hover 시 표시, width: 320px |
| 툴팁 내용 | "DataShot은 매월 5일경 전월 데이터를 정기 업데이트하며, 데이터 품질 향상을 위해 과거 데이터(최근 2년)의 보정 작업이 비정기적으로 진행될 수 있습니다." |

## 10. 에러 처리 및 엣지 케이스

### 10.1 접근 제어

| 시나리오 | 동작 |
|---|---|
| Completed가 아닌 상태의 데이터셋 URL 직접 접근 | 에러 페이지 또는 목록 리다이렉트 |
| 존재하지 않는 ID 접근 | 404 에러 처리 |
| 삭제된 데이터셋 접근 | 에러 안내 + 목록 이동 유도 |

### 10.1A 슬롯 가시성 변경에 따른 상세 페이지 접근 제어

> 본 정책은 사용자 권한(Role)에 영향을 받지 않으며, 오직 '슬롯의 가시성 설정'에 따라서만 제어됨

기존 Private 또는 Internal 슬롯의 가시성이 **External로 변경**된 경우, 해당 슬롯 하위의 **종합 지표 데이터셋**에 대한 사후 방어 로직입니다.

| 항목 | 내용 |
|---|---|
| 적용 대상 | External로 변경된 슬롯 하위에 기존에 생성되어 있던 종합 지표 데이터셋 |
| 적용 시점 | ① 데이터셋 목록에서 클릭하여 상세 페이지 진입 시도 시 ② 알림 목록 데이터셋 생성 완료 알림 클릭 시 |
| 유저 안내 | 화면 우측 상단에 **Error Toast** 컴포넌트 노출 (전체 사용자 공통) |

#### Error Toast 스펙

| 요소 | 스펙 |
|---|---|
| 타입 | error |
| 아이콘 | AlertTriangle (red) |
| 타이틀 | "조회 제한" |
| 내용 | "종합 지표 데이터셋은 External Slot에서 조회가 불가능합니다. 데이터셋을 Private 또는 Internal Slot으로 이동하거나 관리자에게 문의해 주세요." |
| 자동 닫힘 | 5초 |
| 동작 | 상세 페이지 진입 차단 (목록 화면 유지) |

### 10.2 만료 상태

| 시나리오 | 동작 |
|---|---|
| Expired 데이터셋 접근 시도 | "만료된 데이터셋입니다. 동일한 조회 조건으로 재추출하려면 데이터셋을 복사하여 새로 생성해 주세요." 안내 |

## 11. 접근성 고려사항

| 항목 | 기준 |
|---|---|
