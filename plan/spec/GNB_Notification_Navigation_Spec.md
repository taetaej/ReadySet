# GNB 알림 센터 — 알림 클릭 네비게이션 정책 명세서

> 목적: QA 테스트케이스(TC) 작성 기준 문서  
> 버전: v1.0  
> 작성일: 2026-05-26  
> 범위: GNB 알림 센터에서 알림 클릭 시 랜딩 및 유효성 검증 로직

---

## 1. 개요

GNB 알림 센터의 알림 항목을 클릭했을 때, 대상 리소스(시나리오/데이터셋)의 유효성을 단계적으로 검증하고 적절한 페이지로 랜딩하는 정책을 정의한다.

---

## 2. 시나리오 생성 완료 알림 클릭

### 2.1 검증 흐름 (3단계)

```
[알림 클릭]
    │
    ▼
[1단계] 슬롯 ID 일치 여부 검증
    │
    ├─ [YES] 슬롯 ID 일치 (기존 슬롯)
    │       │
    │       ▼
    │   [2단계] 상위 슬롯 유효성 검증
    │       │
    │       ├─ [NO] 슬롯 삭제/권한 없음 → SlotBoard 이동 + Error 토스트
    │       │
    │       └─ [YES] 슬롯 정상
    │               │
    │               ▼
    │           [3단계] 시나리오 유효성 검증
    │               │
    │               ├─ [NO] 시나리오 삭제됨 → 빈 상세 1~2초 → 목록 리다이렉트 + Error 토스트
    │               │
    │               └─ [YES] 시나리오 존재 → 시나리오 상세 페이지 이동
    │
    └─ [NO] 슬롯 ID 불일치 (다른 슬롯으로 이동됨)
            │
            ▼
        이동된 신규 슬롯 최종 검증
            │
            ├─ [정상] 신규 슬롯 권한 있음 + 시나리오 존재
            │       → 변경된 슬롯의 시나리오 상세 페이지 이동 + Info 토스트
            │
            ├─ [예외1] 신규 슬롯 삭제/권한 없음
            │       → SlotBoard 이동 + Error 토스트
            │
            └─ [예외2] 시나리오 최종 삭제됨
                    → 빈 상세 1~2초 → 해당 슬롯 시나리오 목록 리다이렉트 + Error 토스트
```

### 2.2 1단계: 슬롯 ID 일치 여부 검증

| 조건 | 설명 |
|---|---|
| 비교 대상 | 알림의 슬롯 ID (생성 시점) vs 시나리오의 현재 슬롯 ID |
| YES | 동일 슬롯에 위치 → 2단계 진입 |
| NO | 다른 슬롯으로 이동됨 → 이동된 슬롯 검증 |

### 2.3 2단계: 상위 슬롯(Slot) 유효성 검증

| 조건 | 결과 | 랜딩 | 토스트 |
|---|---|---|---|
| 슬롯 삭제됨 | 즉시 리다이렉트 | SlotBoard (`/slotboard`) | Error |
| 슬롯 권한 없음 | 즉시 리다이렉트 | SlotBoard (`/slotboard`) | Error |
| 슬롯 정상 | 3단계 진입 | - | - |

| 토스트 상세 |  |
|---|---|
| 타입 | Error |
| 타이틀 | 접근 불가 항목 |
| 내용 | "Slot이 삭제되었거나 접근 권한이 없습니다." |

### 2.4 3단계: 시나리오 유효성 검증

| 조건 | 결과 | 랜딩 | 토스트 |
|---|---|---|---|
| 시나리오 삭제됨 | 빈 상세 1~2초 노출 후 리다이렉트 | 해당 슬롯의 시나리오 목록 페이지 | Error |
| 시나리오 존재 | 정상 이동 | 시나리오 상세 페이지 | 없음 |

| 토스트 상세 (삭제됨) |  |
|---|---|
| 타입 | Error |
| 타이틀 | 존재하지 않는 항목 |
| 내용 | "삭제되었거나 존재하지 않는 시나리오입니다." |

### 2.5 슬롯 이동 케이스 (1단계 NO)

#### 정상: 이동한 슬롯 권한 있음 + 시나리오 존재

| 항목 | 스펙 |
|---|---|
| 랜딩 | 현재 변경된 슬롯 내의 시나리오 상세 페이지 |
| 브레드크럼 주의 | 상단 브레드크럼 슬롯명에 '이전 슬롯명'이 1초 정도 노출될 수 있음 |
| 토스트 타입 | Info |
| 토스트 타이틀 | Slot 이동 안내 |
| 토스트 내용 | "'{현재 Slot명}' Slot에 위치한 시나리오 상세 페이지입니다." |

#### 예외1: 이동한 신규 슬롯 삭제/권한 없음

| 항목 | 스펙 |
|---|---|
| 랜딩 | SlotBoard (`/slotboard`) |
| 토스트 타입 | Error |
| 토스트 타이틀 | 접근 불가 항목 |
| 토스트 내용 | "Slot이 삭제되었거나 접근 권한이 없습니다." |

#### 예외2: 이동 후 시나리오 최종 삭제됨

| 항목 | 스펙 |
|---|---|
| 랜딩 (1차) | 데이터가 비어있는 상세 페이지 1~2초 노출 |
| 랜딩 (2차) | 해당 슬롯 하위의 시나리오 목록 페이지로 강제 리다이렉트 |
| 토스트 타입 | Error |
| 토스트 타이틀 | 존재하지 않는 항목 |
| 토스트 내용 | "삭제되었거나 존재하지 않는 시나리오입니다." |

## 3. 시나리오 생성 실패 알림 클릭

### 3.1 검증 흐름 (1단계)

```
[알림 클릭]
    │
    ▼
[1단계] 상위 슬롯(Slot) 유효성 검증
    │
    ├─ [NO] 슬롯 삭제/권한 없음
    │       → SlotBoard 이동 + Error 토스트
    │
    └─ [YES] 슬롯 정상
            → 해당 슬롯 하위의 시나리오 목록 페이지로 이동
```

### 3.2 상세

| 조건 | 결과 | 랜딩 | 토스트 |
|---|---|---|---|
| 슬롯 삭제/권한 없음 | 즉시 리다이렉트 | SlotBoard (`/slotboard`) | Error |
| 슬롯 정상 | 정상 이동 | 해당 슬롯의 시나리오 목록 페이지 | 없음 |

| 토스트 상세 (슬롯 무효) |  |
|---|---|
| 타입 | Error |
| 타이틀 | 접근 불가 항목 |
| 내용 | "Slot이 삭제되었거나 접근 권한이 없습니다." |

## 4. URL 직접 접근 시 검증 정책

알림 클릭이 아닌, 사용자가 URL을 직접 입력하여 접근하는 경우에도 동일한 검증 로직을 적용한다.

### 4.1 결과물이 삭제됐거나 없는 ID로 접근

| 항목 | 스펙 |
|---|---|
| 랜딩 (1차) | 데이터가 비어있는 상세 페이지 1~2초 노출 |
| 랜딩 (2차) | 해당 슬롯 하위의 시나리오(또는 데이터셋) 목록 페이지로 이동 |
| 예외 | 슬롯마저 무효한 경우 → 4.3 규칙 적용 |
| 토스트 타입 | Error |
| 토스트 타이틀 | 존재하지 않는 항목 |
| 토스트 내용 | "삭제되었거나 존재하지 않는 시나리오입니다." (또는 "...데이터셋입니다.") |

### 4.2 결과물이 다른 슬롯으로 이동된 상태에서 이전 URL로 접근

| 항목 | 스펙 |
|---|---|
| 랜딩 | 현재 변경된 새로운 슬롯의 상세 페이지로 이동 |
| 토스트 타입 | Info |
| 토스트 타이틀 | Slot 이동 안내 |
| 토스트 내용 | "'{현재 Slot명}' Slot에 위치한 시나리오 상세 페이지입니다." |

### 4.3 슬롯이 삭제됐거나 권한 없는 슬롯 ID로 접근

| 항목 | 스펙 |
|---|---|
| 랜딩 | SlotBoard (`/slotboard`) |
| 토스트 타입 | Error |
| 토스트 타이틀 | 접근 불가 항목 |
| 토스트 내용 | "Slot이 삭제되었거나 접근 권한이 없습니다." |

## 5. 토스트 메시지 정리

| # | 상황 | 타입 | 타이틀 | 내용 |
|---|---|---|---|---|
| 1 | 슬롯 삭제/권한 없음 | Error | 접근 불가 항목 | "Slot이 삭제되었거나 접근 권한이 없습니다." |
| 2 | 시나리오(데이터셋) 삭제/미존재 | Error | 존재하지 않는 항목 | "삭제되었거나 존재하지 않는 시나리오입니다." |
| 3 | 시나리오(데이터셋) 슬롯 이동됨 | Info | Slot 이동 안내 | "'{현재 Slot명}' Slot에 위치한 시나리오 상세 페이지입니다." |

## 6. 랜딩 규칙 요약

| # | 조건 | 랜딩 페이지 | 중간 상태 |
|---|---|---|---|
| 1 | 슬롯 정상 + 시나리오 존재 | 시나리오 상세 페이지 | 없음 |
| 2 | 슬롯 정상 + 시나리오 삭제 | 해당 슬롯 시나리오 목록 | 빈 상세 1~2초 노출 후 리다이렉트 |
| 3 | 슬롯 삭제/권한 없음 | SlotBoard | 즉시 리다이렉트 (하위 검증 생략) |
| 4 | 슬롯 이동됨 + 신규 슬롯 정상 + 시나리오 존재 | 신규 슬롯의 시나리오 상세 | 브레드크럼 이전 슬롯명 1초 노출 가능 |
| 5 | 슬롯 이동됨 + 신규 슬롯 삭제/권한 없음 | SlotBoard | 즉시 리다이렉트 |
| 6 | 슬롯 이동됨 + 시나리오 삭제 | 신규 슬롯 시나리오 목록 | 빈 상세 1~2초 노출 후 리다이렉트 |
| 7 | 생성 실패 알림 + 슬롯 정상 | 해당 슬롯 시나리오 목록 | 없음 |
| 8 | 생성 실패 알림 + 슬롯 삭제/권한 없음 | SlotBoard | 즉시 리다이렉트 |

## 7. 비즈니스 규칙 검증 포인트 (TC 작성 기준)

| # | 검증 항목 | 기대 결과 |
|---|---|---|
| 1 | 완료 알림 클릭 → 슬롯 정상 + 시나리오 존재 | 시나리오 상세 페이지 정상 랜딩 |
| 2 | 완료 알림 클릭 → 슬롯 삭제됨 | SlotBoard 이동 + "접근 불가 항목" Error 토스트 |
| 3 | 완료 알림 클릭 → 슬롯 권한 없음 | SlotBoard 이동 + "접근 불가 항목" Error 토스트 |
| 4 | 완료 알림 클릭 → 슬롯 정상 + 시나리오 삭제 | 빈 상세 1~2초 → 시나리오 목록 리다이렉트 + "존재하지 않는 항목" Error 토스트 |
| 5 | 완료 알림 클릭 → 시나리오가 다른 슬롯으로 이동됨 + 신규 슬롯 정상 | 신규 슬롯 상세 이동 + "Slot 이동 안내" Info 토스트 |
| 6 | 완료 알림 클릭 → 시나리오 이동됨 + 신규 슬롯 삭제 | SlotBoard 이동 + "접근 불가 항목" Error 토스트 |
| 7 | 완료 알림 클릭 → 시나리오 이동됨 + 시나리오 삭제 | 빈 상세 1~2초 → 신규 슬롯 목록 리다이렉트 + "존재하지 않는 항목" Error 토스트 |
| 8 | 실패 알림 클릭 → 슬롯 정상 | 시나리오 목록 페이지 이동 |
| 9 | 실패 알림 클릭 → 슬롯 삭제/권한 없음 | SlotBoard 이동 + "접근 불가 항목" Error 토스트 |
| 10 | URL 직접 접근 → 존재하지 않는 시나리오 ID | 빈 상세 1~2초 → 목록 리다이렉트 + "존재하지 않는 항목" Error 토스트 |
| 11 | URL 직접 접근 → 이동된 시나리오의 이전 URL | 신규 슬롯 상세 이동 + "Slot 이동 안내" Info 토스트 |
| 12 | URL 직접 접근 → 삭제된 슬롯 ID | SlotBoard 이동 + "접근 불가 항목" Error 토스트 |
| 13 | 빈 상세 페이지 노출 시간 | 1~2초 후 자동 리다이렉트 (즉시 아님) |
| 14 | 슬롯 이동 시 브레드크럼 | 이전 슬롯명이 약 1초 노출 후 현재 슬롯명으로 갱신 |
| 15 | 하위 검증 생략 규칙 | 슬롯 무효 시 시나리오 검증 없이 즉시 SlotBoard 이동 |
| 16 | 토스트 자동 닫힘 | 3~5초 후 자동 사라짐 |
| 17 | 데이터셋 삭제 URL 접근 | "삭제되었거나 존재하지 않는 데이터셋입니다." 토스트 + 목록 리다이렉트 |
