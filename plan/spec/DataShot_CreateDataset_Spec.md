## 2. 레이아웃 구조

### 2.1 전체 구조

```
┌─────────────────────────────────────────────────────────────┐
│ AppLayout (GlobalNavBar + Sidebar + Breadcrumb)              │
├─────────────────────────────────────────────────────────────┤
│ 페이지 헤더: "새 데이터셋 생성" + 설명                        │
├─────────────────────────────────────────────────────────────┤
│ ┌──────────────────┬───┬──────────────────┐                 │
│ │ 좌측 (800px)     │ | │ 우측 (420px)     │                 │
│ │ - 스테퍼         │ | │ Configuration    │                 │
│ │ - 스텝 컨텐츠    │ | │ Summary          │                 │
│ │ - 네비게이션 버튼 │ | │                  │                 │
│ └──────────────────┴───┴──────────────────┘                 │
└─────────────────────────────────────────────────────────────┘
```

- Grid: `800px | 1px(구분선) | 420px`, gap: 48px
- 최대 너비: 1400px, 중앙 정렬
- 패딩: 32px

|---|---|---|
| 미입력 | trim() === '' | "데이터셋명을 입력해주세요." |
| 공백만 입력 | length > 0 && trim() === '' | "공백만으로 구성할 수 없습니다." |

| Internal | O | O | 두 옵션 모두 활성화 |
