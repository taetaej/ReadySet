# 프로젝트 현황 요약 (은서님용)

## 📊 프로젝트 개요

### 프로젝트명
**예측/분석 플랫폼 (ReadySet Platform)**

### 목적
광고 캠페인의 예측 및 분석을 위한 통합 플랫폼

### 기술 스택
- **프론트엔드**: React 18.2.0 + TypeScript 5.0.2
- **라우팅**: React Router DOM 7.13.0
- **빌드 도구**: Vite 4.4.5
- **아이콘**: Lucide React 0.263.1
- **차트**: ECharts 6.0.0, Recharts 3.7.0
- **날짜**: date-fns 4.1.0, React Day Picker 9.13.0

---


## ✅ 구현 완료된 기능

### 1. 공통 레이아웃 시스템
- **GNB (Global Navigation Bar)**: 상단 네비게이션
- **SNB (Side Navigation Bar)**: 좌측 사이드바
- **Breadcrumb**: 경로 표시
- **Footer**: 하단 푸터
- **다크모드**: 자동 지원

### 2. SlotBoard (워크스페이스)
- Slot 목록 표시 (그리드 뷰)
- 검색/필터/정렬 기능
- Slot 생성/수정/삭제
- 페이지네이션

### 3. Reach Caster (완료)
- Slot 상세 페이지
- 시나리오 목록 (List/Timeline 뷰)
- 시나리오 생성 (3단계 위자드)
  - Step 1: 기본 정보
  - Step 2: 상세 설정 (Ratio Finder / Reach Predictor)
  - Step 3: 검토 및 실행
- 시나리오 결과 화면
  - Ratio Finder 결과
  - Reach Predictor 결과

### 4. 공통 컴포넌트
- Avatar (프로필 아바타)
- SplashCursor (커서 효과)
- SplitText (텍스트 애니메이션)
- SlotHeader (Slot 정보 헤더)
- DateRangePicker (날짜 범위 선택)

## 🔗 라우팅 구조

```typescript
// App.tsx
<Routes>
  <Route path="/" element={<SlotBoardLayout />} />
  <Route path="/slotboard" element={<SlotBoardLayout />} />
  <Route path="/reachcaster" element={<SlotBoardLayout initialView="slotDetail" />} />
  <Route path="/reachcaster/scenario/new" element={<CreateScenario />} />
  <Route path="/reachcaster/scenario/ratio-finder/result" element={<RatioFinderResult />} />
```
