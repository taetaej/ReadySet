# Validation Message Policy

폼 유효성 검사 메시지의 표시 방식, 위치, 스타일을 통일하기 위한 정책.

## 1. 메시지 스타일 토큰

| 속성 | 값 | 비고 |
|------|-----|------|
| fontSize | `12px` | 플랫폼 전체 통일 |
| color | `hsl(var(--destructive))` | CSS 변수 사용 |
| marginTop | `4px` | 대상 필드 바로 아래 |
| fontWeight | 기본(400) | 강조 불필요 |
| 아이콘 | 없음 | 텍스트만 표시 |
| 배경색 | 없음 | 배경 없이 텍스트만 |

## 2. 메시지 표시 위치 규칙

### 2-1. 필드 레벨 (인라인)
- 해당 입력 필드 바로 아래에 표시
- `marginTop: '4px'`로 필드와 간격 유지
- 필드가 grid 내부에 있으면 grid 밖(아래)에 배치

```
[입력 필드]
유효성 메시지          ← 여기
```

### 2-2. 그룹 레벨 (섹션 하단)
- 라디오/체크박스 그룹, 버튼 그룹 등 복수 선택 UI의 경우
- 그룹 전체 아래에 표시
- `marginTop: '8px'`

```
[버튼A] [버튼B] [버튼C]
유효성 메시지          ← 여기
```

### 2-3. 합계 레벨 (비중 배분 전용)
- 비중 합계가 100%가 아닐 때 합계 표시 바로 아래
- 합계 숫자 색상도 `hsl(var(--destructive))`로 변경

```
합계                    75%  ← 빨간색
비중 합계가 75%입니다. 100%로 맞춰주세요.
```

### 2-4. 0% 비중 경고
- 해당 항목 행 바로 아래 (grid 밖)
- 상위(매체): "비중이 0%입니다. 값을 입력하거나 매체를 선택 해제해주세요."
- 하위(상품/채널): "비중이 0%입니다. 값을 입력하거나 삭제해주세요."
- 미입력(`undefined`)과 `0` 모두 동일하게 처리

### 2-5. 카테고리/탭 레벨
- 탭 영역 바깥 하단에 표시
- 여러 메시지가 있을 경우 `marginTop: '4px'`로 간격

## 3. 입력 필드 border 변경 규칙

| 대상 | 조건 | border 색상 |
|------|------|-------------|
| text input | 미입력/유효하지 않음 | `hsl(var(--destructive))` |
| select/dropdown | 미선택 | `hsl(var(--destructive))` |
| 버튼 그룹 (라디오형) | 미선택 | 각 버튼 border → `hsl(var(--destructive))` |
| 라디오 버튼 카드 | 미선택 | 카드 border → `hsl(var(--destructive))` |
| 상품 목록 컨테이너 | 비중 합계 ≠ 100% | 컨테이너 border → `hsl(var(--destructive))` |
| 날짜 선택기 | 미선택 | `hasError` prop → border 변경 |

## 4. 유효성 검사 활성화 시점

- 사용자가 첫 입력을 시작하면 `validationActive = true`
- 디폴트 값(예: 타겟 GRP 전체 선택)은 사용자 입력으로 간주하지 않음
- `validationActive`가 `true`인 경우에만 메시지 표시
- 비중 0% 경고는 `validationActive`와 무관하게 항상 표시 (선택된 상태에서 0이면 즉시 안내)

## 5. "다음" 버튼 비활성화 조건

- 필수 필드 미입력
- 비중 합계 ≠ 100%
- 비중 0% 항목 존재 (미입력 포함)
- 카테고리별 최소 선택 미충족
- 상품/채널 미선택 매체 존재

## 6. 메시지 문구 패턴

| 유형 | 패턴 | 예시 |
|------|------|------|
| 필수 입력 | `{필드명}을/를 입력해주세요.` | "총 예산을 입력해주세요." |
| 필수 선택 | `{필드명}을/를 선택해주세요.` | "매체를 선택해주세요." |
| 비중 합계 | `비중 합계가 {N}%입니다. 100%로 맞춰주세요.` | |
| 비중 0% (상위) | `비중이 0%입니다. 값을 입력하거나 {해제 액션}해주세요.` | "...매체를 선택 해제해주세요." |
| 비중 0% (하위) | `비중이 0%입니다. 값을 입력하거나 삭제해주세요.` | |
| 최소 선택 | `최소 {N}개 이상의 {대상}을/를 선택해주세요.` | |
| 범위 오류 | `{조건} 확인해주세요.` | "최솟값은 최댓값보다 작아야 합니다." |
